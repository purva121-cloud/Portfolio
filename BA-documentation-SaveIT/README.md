# SaveIT – Business Analysis Case Study

An end-to-end Business Analysis portfolio project for **SaveIT**, a personal finance management application. This project demonstrates the Business Analyst's role throughout the Software Development Life Cycle (SDLC), from understanding the business problem to preparing project documentation, Agile artifacts, and validation deliverables.

## Project Artifacts

* Project Charter
* Business Analyst Case Study
* Business Requirements Document (BRD)
* Stakeholder Power–Interest Matrix
* Agile Artifacts (Product Backlog, User Stories & Sprint Planning)
* Test Cases & User Acceptance Testing (UAT)

## Tools Used

* Microsoft Word
* Microsoft Excel
* Lucidchart

## Methodology

* SDLC
* Agile Scrum
* MVP Approach
