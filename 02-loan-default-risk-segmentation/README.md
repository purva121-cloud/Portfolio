# 💳 Loan Segmentation & Risk Analysis

A data analytics project focused on analyzing loan applicants, identifying customer segments, and assessing credit risk using SQL, Excel, and Power BI.

## Overview
This project explores loan applicant data to uncover patterns in borrower behavior, evaluate risk profiles, and support data-driven lending decisions through interactive dashboards and analytical insights.

## Features
- Customer Segmentation
- Credit Risk Analysis
- Loan Approval Insights
- Default Risk Assessment
- Interactive KPI Dashboard
- Business Recommendations

## Tools Used
- SQL
- Power BI
- Microsoft Excel
- Power Query
