# Retail Inventory Optimization

An end-to-end Inventory Analysis project focused on optimizing inventory performance using SQL, Excel, and Power BI.

## Overview
This project analyzes inventory data to identify stock health, reorder requirements, fast and slow-moving products, and key business KPIs through interactive dashboards.

## Tools Used
- SQL
- Power BI
- Microsoft Excel
- Power Query
